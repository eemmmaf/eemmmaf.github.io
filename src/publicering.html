Publicering

Den publicerings-plattform jag har valt att använda är Github Pages. Jag har aldrig tidigare testat den publiceringsplattformen. Den enda publiceringsplattformen jag har testat innan är Netlify, bortsett från publiceringsplattformar som är baserad på FTP/SFTP.

Om verktyget

Github Pages är en publiceringsplattform som gör det möjligt att publicera en webbplats direkt från ens Github-repository. Det går att ladda upp statiska sidor med HTML, CSS och JavaScript. Alla sidor som är skapade efter 2016-06-15 och har domänen github.io använder HTTPS.


Länk till publiceringsverktyget

Stöd för programmeringsspråk

HTML, CSS och JavaScript

Priser

Att ladda upp en webbplats via Github Pages är gratis. Det finns däremot olika pris-planer för Github där mer funktioner ingår i de planerna som kostar. Jag använder versionen som är gratis och kunde utan problem ladda upp en webbplats via Github Pages.

Fördelar

  • Det är gratis att använda
  • Det går att redigera koden direkt från Github, vilket gör det smidigt om man vill ändra något snabbt
  • Obegränsat antal sidor för projekt-sidor
  • Det finns bra guider som Github erbjuder
  • Github erbjuder Jekyll, en generator som kan generera en statisk webbplats. Det har jag inte hunnit testa än dock.
  • Enkelt att dra och släppa filer som ska publiceras

Nackdelar

  • Det är ej tillåtet att använda för att driva ens online-företag.
  • Begränsad storlek på webbplatserna. En webbplats får inte vara större än 1GB.
  • Begränsad bandbreddsgräns på 100 GB/månad
  • Endast en site per användare

Använda Github Pages

För att använda mig av Github Pages har jag först skapat ett repository med mitt Github-användarnamn. Sedan har jag klonat det repot till min dator. Efter det skapade jag en index.html-fil. Till sist användes kommandona git add, git commit och git push för att lägga till det i repot. Jag har även testat att dra och släppa filer till repot via filuppladdningen. Jag har även testat att redigera en fil direkt i repot när jag skulle skriva en kommentar. Det är första gången jag använder Github Pages och jag tycker att det var supersmidigt.
